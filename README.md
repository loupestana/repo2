# WHat's my name

This is an example project demonstrating how to publish a python module to PyPI.



## Installation

Run the following to install :

```python
pip install Whats_My_Name
```


## Usage

```python 
from Whats_My_Name import whats_my_name 

#Generate "Rui Pires"
whats_my_name(Rui)

#Generate "Other"
whats_my_name(other)
```
